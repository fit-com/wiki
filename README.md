## ✍️ Хочешь добавить свою историю?

Мы пишем этот лор вместе. Если у тебя есть крутая история, мем или описание препода — не молчи. 

**Инструкция для авторов:**
1. Нажми кнопку `Add file` -> `Create new file` (если хочешь создать новую статью) ИЛИ открой существующую статью и нажми на иконку карандаша `Edit this file`.
2. Назови файл латиницей, например: `moya-istoriya.md`.
3. Напиши текст в формате Markdown.
4. Внизу страницы выбери пункт **"Create a new branch for this commit and start a pull request"**.
5. Нажми зеленую кнопку **Propose changes**.
6. Админы проверят текст на адекватность и добавят его на сайт!
